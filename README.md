# Terraform AWS EC2 Assignment

## 📌 Objective

This project demonstrates how to use Terraform to provision an AWS EC2 instance using Infrastructure as Code (IaC).

---

## 🛠️ Technologies Used

* Terraform
* AWS (EC2)

---

## ⚙️ Steps to Run

1. Initialize Terraform

```
terraform init
```

2. Preview Execution Plan

```
terraform plan
```

3. Apply Configuration

```
terraform apply
```

---

## 📦 Resources Created

* EC2 Instance
* Amazon Linux AMI
* Instance Type: t2.micro
* Tag: Terraform-Student-Instance

---

## 📤 Output

* Public IP of EC2 instance

---

## 🧹 Cleanup

To avoid charges:

```
terraform destroy
```
